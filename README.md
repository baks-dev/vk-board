# BaksDev Vk Board

[![Version](https://img.shields.io/badge/version-7.3.0-blue)](https://github.com/baks-dev/vk-board/releases)
![php 8.4+](https://img.shields.io/badge/php-min%208.4-red.svg)
[![packagist](https://img.shields.io/badge/packagist-green)](https://packagist.org/packages/baks-dev/vk-board)

Модуль Vk Board

## Установка

``` bash
$ composer require baks-dev/vk-board
```

## Дополнительно

Установка конфигурации и файловых ресурсов:

``` bash
$ php bin/console baks:assets:install
```

## Тестирование

``` bash
$ php bin/phpunit --group=vk-board
```
